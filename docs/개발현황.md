## 🐛 해결한 오류 목록

| 오류 | 원인 | 해결 방법 |
|------|------|-----------|
| 404 (Pages 안 켜짐) | GitHub Pages 설정 미완료 | Settings → Pages → Deploy from a branch → main |
| Content-Encoding: br | Unity Brotli 압축 | Compression Format → Disabled |
| Content-Encoding: gzip | Unity Gzip 압축 | Compression Format → Disabled |
| .gz 404 + Invalid token | 이전 빌드 파일 잔여 | games 폴더 완전 삭제 후 새 빌드 복사 |
| Changes 0 감지 | Git 이 변경 미감지 | git add -f games/ |
| 브라우저 캐시 | 오래된 파일 로드 | Ctrl + Shift + R 강력 새로고침 |

## 💡 배운 것 / 팁

1. **Unity WebGL 빌드시 Compression Format 은 반드시 Disabled**
   - GitHub Pages 는 Brotli/Gzip 압축 해제를 자동으로 하지 않음

2. **브라우저 캐시는 강력 새로고침으로 해결**
   - `Ctrl + Shift + R` (Windows) / `Cmd + Shift + R` (Mac)

3. **GitHub Pages 는 main 브랜치를 기본으로 사용**
   - master 브랜치는 환경 보호 규칙으로 배포 거부됨

4. **개인정보는 공개 사이트에 직접 노출 금지**
   - 이메일 대신 GitHub Issues, 연락처 폼 사용

5. **파일 교체 시 기존 파일 완전 삭제 후 복사**
   - 이전 빌드 파일이 섞이면 예측 불가능한 오류 발생
